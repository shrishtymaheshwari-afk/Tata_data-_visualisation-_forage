# Tata_data-_visualisation-_forage
 ### Project Overview
This project is part of the Tata Group Data Visualisation Job Simulation hosted on Forage. The simulation replicates real-world data analyst tasks at Tata Insights and Quants (Tata iQ).

## Tools Used

Tableau — for dashboard creation and data visualisation
Excel — for data exploration


## Tasks Completed
Task 1 — Framing the Business Scenario
Analysed business requirements from the perspective of the CEO and CMO of an online retail company. Framed the right questions before touching the data.

Task 2 — Choosing the Right Visuals
Identified which chart types best communicate different kinds of data — avoiding misleading or ineffective visualisations.

Task 3 — Creating Effective Visuals
Built an interactive Tableau dashboard showing:
-Month vs Revenue trend (Dec 2010 – Dec 2011)
-Country-wise quantity map
-Top 10 countries by revenue and quantity
-Top 10 customers by revenue

Task 4 — Communicating Insights and Analysis
Translated complex data findings into clear, actionable insights for senior leadership.

## Key Insights from Dashboard

-November 2011 had the highest revenue — ₹14,61,756
-Netherlands was the top country by revenue
-Customer 14646 was the highest revenue-generating customer
